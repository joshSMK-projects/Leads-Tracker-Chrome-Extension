Project Name
    Leads Chrome Extension

Project Description
    This project allows users to store important urls for further use, similar to bookmarks.
    But unlike bookmarks we can also input our own url and delete all of the leads.

How to run the project
    1. Download the project
    2. Go to chrome://extensions/
    3. Trigger Developer mode on
    4. Go to Load unpacked and select the folder you downloaded
    5. Go to extensions widget (puzzle icon) and scroll down to find Leads trackers and select it

How to use the project
    There are two ways to save a url:
        1. Copying and Pasting through the text input and "SAVE INPUT" button
        2. Clicking the "SAVE TAB" button which grabs the url of the current page
    
    To delete all urls double-click the "DELETE ALL" button
